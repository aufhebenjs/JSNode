在你的 Express.js 应用中，你使用了 res.json('你好') 来发送 JSON 响应。
然而，res.json 方法实际上是用于将对象或数组转换为 JSON 格式的字符串并发送给客户端。
但在你的例子中，你传递了一个字符串 '你好' 给 res.json。

由于你传递的是一个字符串而不是对象或数组，res.json 在处理时可能会导致一些意外行为。
在这种情况下，它可能将字符串 '你好' 转换为 JSON 格式，并将其发送给客户端。

你在控制台中输出 let s= res.json('你好')，但请注意，res.json 并不返回 JSON 字符串，
而是在内部将 JSON 发送到客户端，并且没有返回值。
因此，let s= res.json('你好') 中的 s 实际上是 undefined。
